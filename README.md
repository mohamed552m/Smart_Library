# Smart Library

Smart Library is an Android UI project for a smart library application.

It is designed to create a smooth and simple reading experience where users can browse books, track their borrowings, and manage their profile through a clean and modern interface.

## Project Overview

The main idea behind this project is to build a visually organized library app with a consistent mobile design system.

The UI is designed to be:
- clean
- simple
- readable
- modern
- easy to navigate

The visual identity of the app is based on a dark top header, light backgrounds, soft gray cards, and rounded components.

## Screens

- **Splash**
- **Home**
- **Books**
- **Borrowings**
- **Profile**

## Key Highlights

- Modern Android UI
- Consistent visual style across all screens
- Clean mobile layout
- Dark header with light content area
- Soft gray cards and rounded corners
- Book browsing and borrowing flow
- Profile section with editable user information
- Simple and clear bottom navigation

## Tech Stack

- **Kotlin**
- **Android XML**
- **Fragments**
